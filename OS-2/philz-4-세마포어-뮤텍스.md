# 세마포어 & 뮤텍스

# 뮤텍스

한 쓰레드, 프로세스에 의해 소유될 수 있는 Key를 기반으로 한 상호 배제 기법

임계 구역을 가진 쓰레드들의 실행시간이 서로 겹치지 않고 각각 단독으로 실행되게 하는 기술

# 세마포어

현재 공유 자원에 접근할 수 있는 쓰레드, 프로세스의 수를 나타내는 값을 두어 상호배제를 달성하는 기법

세마포어 P, V 연산

S: 자원의 개수
P: 임계 구역 들어가기 전에 수행
V: 임계 구역에서 나올 때 수행

```
P(S): 현재 실행할 수 있는지 확인
V(S): 실행하면서 자원을 점유함을 마킹
```
